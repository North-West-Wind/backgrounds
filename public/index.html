Integrelle Takes a Break ðŸ”—

"Integrelle Takes a Break" supports configuration through search parameters.
For example, https://background.northwestw.in/break/?rain&rainVolume=0.1

  • norain - Make it never rain
  • rain - Make it always rain
  • rainVolume <float> - Set the rain volume between 0 and 1
  • weather [hko:[station]] - Make it rain depending on real world weather
    • Defaults to get weather info from https://wttr.in
    • Supplying "hko" will get weather info from the Hong Kong Observatory instead
    • Supplying "hko:station" will get weather info of a specific station. You can check the station name here
  • weatherCycle <int[,int...]> - Preset the cycle between rain and sunny
    • Each integer is how long a weather condition will last for (in milliseconds)
    • For example, "20000,10000" will make it clear for 20 seconds, rain for 10 seconds, back to clear for 20 seconds and so on
    • If "rain" is supplied as a search parameter, the starting weather will be rain
    • This overrides the "weather" search parameter

Gallery ðŸ”—

"Gallery" supports your own image list through search parameter "list".
By default, it uses the list /gallery/default.txt
So the page without any options is equivalent to
https://background.northwestw.in/gallery/?list=%2Fgallery%2Fdefault.txt
Each line of the list must be a valid URL that points to an image.
